# RULE

## 配色设计指导

基于马卡龙底色与颗粒纹理，采用斜体粗衬线排版，配合 1.5px 边线与小圆角与不对称布局的编排设计。窄中心列、大小字体对比强烈、标题采用粗衬线体、文本正文数据采用非衬线体。

Light Theme (青春画报模式) 主色调模拟具有颗粒感的复古运动画报纸张，文字使用高浓度的墨色。Pastel Peach - #FFD6A5: Primary
background, paper texture base, main section fill. Pastel Yellow - #FDFFB6: Secondary background, sidebar fill,
secondary content blocks. Pastel Pink - #FFADAD: Decorative card background, image mask overlays, accent highlights.
Pastel Green - #CAFFBF: Functional tags, success states, interactive UI element backgrounds. Charcoal Ink - #24211D:
Primary body text, 1.5px structural borders, hard shadows (0-blur), icon strokes. Varsity Green - #14381C: H1-H3 serif
titles, call-to-action (CTA) button text, varsity-style labels. Dark Theme
(深夜赛场模式) 主色调模拟深色吸光布料或橡胶球场，文字反转为高明度的马卡龙色。Ink Black - #1A1816: Primary background,
matte dark texture base. Dark Burgundy - #4A1A1A: Secondary background, featured container fill, card depth layers. Dark
Olive - #383B10: Tertiary background, decorative block partitioning. Varsity Dark Green - #14381C: Dark mode section
background, high-contrast structural blocks. Cream White - #F9F7F2: Primary body text, 1.5px white borders, subtle
structural lines. Pastel Peach - #FFD6A5: H1-H3 serif titles, CTA button background, key navigational highlights. Pastel
Pink - #FFADAD: Accent text, active state indicators, high-priority notification tags. @import
