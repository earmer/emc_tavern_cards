# 360平米复式户型重构方案

---

## 一、纯户型分割

### 1. 原始结构与打通逻辑

这是同层相邻的两套大户型，均为南北通透的板式结构，共享一面东西向的分户墙。

**A户型（西侧，原购入）**：建筑面积约185平米，三室两厅两卫，陆家最初购入时陆时屿尚在幼儿园。

**B户型（东侧，后购入）**：建筑面积约175平米，两室两厅一卫，夏星眠父母出事后，陆家购入此套用于安置夏星眠，彼时两个孩子十岁。

**打通位置**：两套户型之间的分户墙并非承重墙，在装修时于中段开设了一个2.4米宽的通道，形成内部连接。打通处位于两套原客厅的北侧，靠近各自的餐厅区域，这样既保持了两侧客厅的完整性，又让日常走动不必绕行公共走廊。

打通后，原B户型的入户门封闭，仅保留A户型入户门作为唯一出入口。原B户型入户门位置改为储物柜。

---

整体呈"工"字形动线：南侧是采光面，布置起居和卧室；北侧是服务面，布置厨卫和辅助功能；中部是交通核心。

### 3. 房间相对位置详述

**南立面（从西到东）**：主卧阳台 → 主卧 → 主客厅 → 主客厅阳台 → 家庭厅阳台 → 家庭厅 → 陆时屿卧室 → 陆时屿卧室阳台 → 少年区私密间

**北立面（从西到东）**：入户门 → 玄关 → 公卫 → 厨房 → 厨房北阳台 → 打通通道 → 储物间 → 家政间 → 技术工坊 → 夏星眠卧室（杂物）

**西立面**：主卧套房占据整个西侧，书房在其北侧

**东立面**：少年区私密间占据东南角，少年区卫生间、夏星眠卧室在其北侧，少年区卫生间夹在私密区和夏星眠卧室之间

**中部交通**：从入户玄关进入后，向南进入主客厅，向东穿过餐厅和打通通道可达东翼；主卧入口在客厅西侧，书房入口在西翼走廊尽头；东翼的家庭厅南侧是景观面，北侧走廊连接两间卧室和卫生间。


## 二、动线设计（15条）

### 日常基础动线

**1. 入户动线**
入户门 → 玄关（换鞋、放包）→ 西翼走廊 → 主客厅
*玄关设置换鞋凳和挂衣区，避免将室外灰尘带入室内*

**2. 长辈就寝动线**
主客厅 → 西翼走廊 → 主卧门 → 主卧
*主卧门距客厅沙发区约4米，关门后互不干扰*

**3. 少年就寝动线**
家庭厅 → 东翼走廊 → 陆时屿卧室
*从家庭厅到卧室约6米，经过卫生间门口，方便睡前洗漱*

**4. 晨起洗漱动线（少年区）**
陆时屿卧室 → 东翼走廊 → 少年区卫生间（双台盆）
*两人可同时使用洗手台，马桶间和淋浴间独立，互不等待*

**5. 早餐动线**
各卧室 → 餐厅 → 厨房取餐 → 餐厅用餐 → 玄关出门
*餐厅位于动线中心，四人可错峰使用*

### 功能性动线

**6. 烹饪动线**
冰箱（厨房内）→ 备餐台 → 灶台 → 出餐口 → 餐厅
*封闭式厨房隔绝油烟，出餐口正对餐桌*

**7. 家政动线**
脏衣收集（各卫生间）→ 家政间（洗衣烘干）→ 主卧阳台/家政间内（晾晒折叠）→ 各卧室衣柜
*家政间靠近东翼卫生间，减少搬运距离*

**8. 访客动线**
入户门 → 玄关 → 主客厅 → 公共卫生间
*访客活动范围限于西翼公共区，不进入卧室区域*

**9. 书房动线**
主客厅/餐厅 → 西翼走廊 → 书房
*陆父从公共区进入书房，不经过主卧，不打扰陆母休息*

**10. 储物取用动线**
东翼走廊 → 储物间
*换季衣物、行李箱等低频物品集中存放，不占用卧室空间*

### 特殊场景动线

**11. 夜间如厕动线（长辈）**
主卧床 → 主卧卫生间
*套房设计，无需进入公共区域*

**12. 夜间如厕动线（少年）**
陆时屿卧室床 → 卧室门 → 东翼走廊（3米）→ 少年区卫生间
*走廊设置感应夜灯，低亮度不刺眼*

**13. 周末家庭活动动线**
主客厅（日常起居）↔ 家庭厅（影音游戏）
*两个起居空间通过打通通道连接，可分可合*

**14. 怀旧性爱路线**
少年区 → 陆时屿卧室 → 东南衣柜旁暗门 → 私密区

**15. 保姆工作动线**
入户门 → 玄关 → 厨房 → 家政间 → 各区域打扫 → 入户门离开
*非住家保姆，工作范围明确，不进入卧室私密区*

**16. 紧急疏散动线**
任意位置 → 最近走廊 → 入户门
*户型无死角，任何房间到入户门不超过15米*
